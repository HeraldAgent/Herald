# Herald Runbook

## What Herald Is For

Herald tells the operator whether a headline still has edge after publication or whether the market has already fully distributed it.

## Daily Operator Loop

1. Run `npm run dev`.
2. Read the catalyst board for the highest-ranked event.
3. Confirm surprise, half-life, and contamination still support action.
4. Escalate only the headlines that are still inside their edge window.

## What Gets Promoted

- fresh listings, hacks, unlocks, and regulatory updates with real surprise
- structurally important headlines that still sit inside the category half-life

## What Gets Demoted

- recycled rumor loops
- stories with weak sourcing
- headlines that reached the market too late to matter
